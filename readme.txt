A simple plugin to track FastWay parcels using a shortcode.

== Description ==

FastWay Track and Trace allows users to track FastWay parcels by entering the waybill number. The plugin uses a simple shortcode to display the tracking form and results on any page or post. Users can obtain their FastWay API Key from the FastWay API documentation.

Features:
* Track FastWay parcels using a shortcode
* Display tracking information in a table with StatusDescription, Date, and Name
* Add a FastWay logo above the input and button (optional)
* Simple settings page to configure API Key and logo display

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fastway-track-and-trace` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the 'Settings' > 'FastWay Track and Trace' menu to configure the plugin.
4. Add the `[fastway_tracking]` shortcode to any page or post where you want to display the tracking form.

== Frequently Asked Questions ==

= How do I obtain a FastWay API Key? =

You can obtain your API Key from the FastWay API documentation at https://sa.api.fastway.org/v3/Docs/GetAPIKey.
